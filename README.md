# Landing Page
A modern landing page built with React, Vite, Three.js, and GSAP.

## Features

* Responsive design
* Three.js 3D model integration
* GSAP animations
* Fast performance with Vite

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```
